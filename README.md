# 2025 NDSU Design Build Fly Glider
## Components used
- ESP-32 S3
- AltIMU-10 v6
- SAM-M10Q GPS breakout board
- Adafruit PowerBoost 500C
- 250 mAh LiPo battery
- 2x lightweight servos

## Pin map
- GPIO 10 - LED
- GPIO 11 - LED 2
- GPIO 2 - Left servo
- GPIO 4 - Right servo

- I2C (IMU)
- GPIO 8 - SDA IMU
- GPIO 9 - SCL IMU

- UART (GPS)
- TX - GPS TX
- RX - GPS RX
